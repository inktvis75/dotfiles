Emacs Prelude
=============

Make sure that there is no existing Emacs configuration:

```shellsession
$ rm -rf ~/.emacs.d/

$ rm -rf ~/.config/emacs

$ rm -rf ~/.config/doom
```

Install Prelude Emacs:

```shellsession
$ curl -L https://git.io/epre | sh
```

Start Emacs

When the installation is finished, copy the custom files in the personal directory.
